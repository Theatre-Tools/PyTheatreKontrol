# Project Roadmap

### Sub Roadmaps
- [Eos Roadmap](EOS_ROADMAP.md)


## Short-term Goals
- Implement basic OSC communication for Eos consoles.
- Develop a set of validators to ensure correct communication with the console.
- Decide on unified project wide structure for sending and reciving data.



## Long-term Goals

- Expand support to other brands of lighting consoles.
- Implement a more robust testing suite, including integration tests with actual consoles.
